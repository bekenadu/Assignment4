    PsuedoClass

    Is a selector that selects elemnets, a tag under certian conditions of a certain state, like whe they are being hovered they tend to act as if you had applied a class to some part of your html file. We use this selectors to target part of an element that cant be targeted by using simple methods like class or id. we use PsuedoClass selectors to select elementd based on there attributes, state or even relative positions

  • PsuedoClass are keyword that starts with a colol :
  • : Psuedo-Class-name

    • For Example

  • input type="text" checked > br>
  • input:checked { margin : 32px; }
  • now the second line will grow is margin only when the checked button is on
  • so we apply PsuedoClass to tags that are under certian condiotion

    PsuedoElements

    They behave in similar way but they act as if you had added a new html element.

  • They start using double colon ::
  • they dont apply on tags they apply inside the tags
  • For Example
  • P> This is a demo for PsuedoElements p>
  • p::before {content :"hey"}
  • now the second line will add the content "hey" in our p tag in the begining
  • p:: after { content: "hey"}
  • this will add the content "hey" at the last of our p tag

    Flex Layout

    Flex layout is a layout thats in 1D in a ROW OR COLUMN

    Flex is a content based layout which means content first layout methods

    Before the Flexbox Layout module, there were four layout modes

  • Block, for sections in a webpage
  • Inline, for text
  • for two-dimensional table data
  • Positioned, for explicit position of an element

    • The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

    Grid Layout

    Grid Layout items are in 2D ROWS AND COLUMNS

    Grid Layout is a layout based system which means doing the layout comes first unlike the flex

    grid helps you to create the outer layout of a page so you can build a complex responsive design

    To make an HTML element behave as a grid container, you have to set the display property to grid or inline-grid. Grid containers consist of grid items, placed inside columns and rows.
